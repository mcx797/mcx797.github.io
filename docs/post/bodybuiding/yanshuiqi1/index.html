动作总览

分为多个超级组，所有超级组结束为一组.

共进行三组

超级组1

  1. 二头弯举 + 推肩 （沉肩夹背挺胸）12次

    最高点时保持肘部微弯

超级组2

锤式弯举+侧平举

锤式弯举最高点不要过高，侧平举减重量,最高到手臂水平

超级组3

反向二头弯举 + 反向飞鸟

反向二头弯举练小臂，不要抬得过高

反向飞鸟练后束练背，注意肩

超级组4

三头臂屈伸 + 侧平举

超级组5

颈后臂屈伸 + 反向飞鸟

超级组6

窄距俯卧撑
